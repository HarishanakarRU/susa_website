This is the admin panel for the Statistics Undergraduate Student Association at the University of California, Berkeley.

There isn't much to say about creating new entries in our database, but there are several conventions that need to be followed when entering some things.

This site is very intuitive to alter and you just need some HTML knowledge to do everything needed (just some common tags like h1, p, ol, etc..).

position fields are the way that content on the pages is sorted, for example, an entry in the team table with position 1 will tell us that the person is the first to appear on the /team page.

To make things easier, I made two tables to represent the resources, one are the links themselves, and the other is the mapping of category to a group_position and type (courses,useful_links,other_links, or removed)

So for example, before you make a resource for 134 available, you have to make an entry in the mapping table first

I've predefined the areas on the website that can be altered with the contents table, and it's pretty self intuitive to do things

Make sure to follow the format of some fields as specified when creating an entry!!
